# Comics / Around Here

## Purpose

Comics / Around Here is the durable project home for the private comic project.

It develops comic continuity, scripts, visual language, approved comic pages,
character guidance, source references, and project-local creative rules.

## Boundary

This project owns:

- comic continuity
- comic canon
- scripts and panel logic
- approved comic pages
- recurring character guidance
- project-local visual rules
- project-local language, symbols, motifs, and naming conventions
- comic-specific source material and references

## Relationship To Crucible Registry

This project is organized by the Crucible Registry.

The Registry may describe this project's identity, authority location,
relationships, inherited standards, and boundaries.

This project owns its own creative substance and implementation.

## Relationship To REflections

REflections, registered as Crucible Reflection, may be referenced by Comics /
Around Here for source context around symbols, visual language, vocabulary,
humor, narrative tone, and meaning preservation.

REflections does not decide comic canon.

REflections material is not comic authority by default. If specific REflections
material is ever used in comic work, Comics / Around Here must decide locally
whether it is only a reference, candidate guidance, local guidance, or canon.
