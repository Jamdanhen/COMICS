## Workflow Rule

For future multi-panel comic ideas, start with rough blocking after direction is discussed and chosen.

Rough blocking should show:

- panel movement
- character placement
- relative scale
- key objects
- sequence logic

It does not need:

- final perspective
- polished rendering
- detailed texture
- final style
