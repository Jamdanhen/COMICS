# Proposed REflections Reference Boundary

Status: Proposal
Created: 2026-07-02
Source safety model: `C:\Crucible_Registry\REFLECTION_INFLUENCE_SAFETY_MODEL.md`

## Purpose

This proposal defines how Comics / Around Here should treat REflections when
REflections is referenced as source context.

It protects the comic project from accidentally turning REflections source
material into comic canon.

It does not install REflections guidance into Comics / Around Here.

## Current Authority

Comics / Around Here local guidance currently lives in:

- `C:\COMICS\AGENTS.md`
- `C:\COMICS\Source\Canon\comic-workflow-governance.md`

Taskmaster remains the final authority for comic canon, visual identity, tone,
approval, rejection, revision, and completion.

## Core Rule

REflections may be referenced by Comics / Around Here.

REflections does not govern Comics / Around Here.

REflections material is not comic authority by default.

If specific REflections material is used in comic work, Comics / Around Here
must decide locally whether it is only a reference, candidate guidance, local
guidance, or canon.

## Allowed Reference Areas

REflections may be referenced for:

- symbol dictionary
- visual language
- vocabulary
- humor
- narrative tone
- meaning-preservation guidance
- category and classification humor
- drift prevention
- source-preservation habits

## Not Comic Authority By Default

REflections does not automatically decide:

- comic canon
- scripts
- artwork
- character designs
- issue structure
- publication format
- final project identity
- approved comic pages
- visual changes to recurring characters

## Local Boundary Gate

Before REflections material becomes local comic guidance or canon, confirm:

- what specific material would become local comic authority
- whether it is source, reference, candidate guidance, local guidance, or canon
- where it will live inside `C:\COMICS`
- whether it conflicts with existing `Source/Canon`, `Source/Characters`, or
  `Source/Language`
- whether Taskmaster has approved the local use

If any answer is unclear, keep the material parked as source or candidate
guidance.

## Drift Controls

Use these controls when REflections is referenced in comic work:

- Do not use `Comics / Noodle Language` as the comic project name or subtitle.
- Treat Noodle as unresolved unless Taskmaster explicitly decides otherwise.
- Do not turn REflections analysis into comic doctrine without local comic
  approval.
- Do not replace local comic meaning with global defaults.
- Do not redesign Taskmaster, Razorforge, or Codex based on generalized
  symbolic guidance.
- Use the lineup and character files before generating recurring characters.
- Keep generated images as drafts until Taskmaster approves them.
