## Candidate Definition

Zodak's Helmet represents:

**Unresolved classification.**

It marks a known inconsistency, recognized anomaly, or open question that should be preserved until a sufficient framework exists.

## Core Distinction

This symbol is not the same as a noodle.

A noodle is unresolved significance, possible relevance, or not-yet-classified potential.

Zodak's Helmet is more specific:

- the contradiction has already been noticed
- the available category is known to be wrong or insufficient
- the question has enough shape to be preserved
- the correct framework has not yet arrived

Short form:

> Noodle: "This might matter."

> Zodak's Helmet: "This does not fit the available category."

## Not The Core Meaning

The helmet should not primarily mean:

- wisdom
- balance
- neutrality
- observation

Those can orbit the symbol if a scene needs them, but they are not the core.

## Correct Comic Response

When this symbol appears, the intended response is not immediate resolution.

Useful responses:

- Leave it parked.
- Framework insufficient.
- Question remains open.
- Preserve the contradiction.
- Do not force the category.

## Visual Function

Possible visual uses:

- label on a shelf for unresolved classification problems
- stamped file tab for an anomaly that refuses current categories
- museum placard beside an object whose category has been crossed out
- helmet icon on a parked-question drawer
- newspaper image in an "Around Here Chronicle" gag

Avoid using the symbol as a generic mystery icon.

Avoid using it as a generic wisdom/balance icon.

## Relationship To Existing Language

Related existing concepts:

- Noodle: unresolved significance before classification
- Fragile Detail Jar: preserving uncertain detail without over-deciding
- Parked Idea Shelf: future-facing holding place
- Committee Notice: review pending, authority unclear, or classification disputed
- Archive: durable memory once material is approved

Zodak's Helmet is closest to Fragile Detail Jar plus classification pressure, but with one extra trait:

The original category was visibly wrong.
