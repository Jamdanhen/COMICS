# Approved Vocabulary Index

Status: active index

Date: 2026-07-07

Purpose:

Keep the current comic vocabulary findable without flattening the creative mess.

This index does not replace the source files. It points to them and separates
approved symbols, approved characters, working candidates, parked concepts, and
retired or not-approved items.

Use this as a routing map before writing scripts, image prompts, symbol keys,
or character notes.

## Status Rules

Approved:

- Taskmaster has explicitly approved it, or
- it appears in an approved comic page and the approval notes say inclusion
  approves the symbol, or
- it is recorded in an approved canon/language file as approved.

Working candidate:

- Useful, recurring, or likely, but still needs a clearer visual form,
  stronger context, or explicit promotion.

Parked / observe-only:

- Interesting enough to preserve, but not ready for active comic use.

Retired / not approved:

- Rejected, replaced, superseded, or specifically not approved for current
  vocabulary use.
